Open-Source Secure Node-ReD

We have transitioned our powerful Node-RED deployment script into a fully open-source project. This solution allows you to deploy a secure, embeddable instance of Node-RED, perfect for any IoT or automation project.

  • Embedded with Express.js: Easily integrate Node-RED into your existing Node.js applications.
  • Enhanced Security with TOTP: We've implemented Time-based One-Time Password (TOTP) for robust two-factor authentication, ensuring your flows are always protected.
  • Deploy with Docker: Get up and running in minutes. Our solution is containerized for easy deployment on any platform that supports Docker.

Find the complete source code and deployment instructions on our GitHub repository:node-red-express-totp
